<h1 align="center">Cloud Music Demo</h1>

> web版的云音乐播放器具有基础功能的demo，仿自网易云音乐WindowsPc客户端(version.2.9.5)

## 演示传送门 

[Cloud Music Demo](http://101.34.166.157/)

## 主要技术栈

- Vue3
- Vite
- Typescript
- Pinia
- Axios
- Element-plus
- Sass

## 网易云音乐API

[NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi)
